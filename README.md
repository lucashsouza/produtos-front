# produtos-front

## Project setup
```
npm install
npm install -g @vue/cli @vue/cli-service-global
npm install axios --save
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
